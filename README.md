## Instalation

#### Windows (Using Installer)
1. Go to releases and download "Setup-x86-win.exe".
2. Run "Setup-x86-win.exe" and basicly install it.
3. Run the installed ChattyApp.
4. Have fun!

#### Linux
1. Go to releases and download the binary "ChattyApp.py".
2. Open terminal.
3. Install python and the some library (Python 3.8+, PyQt5, PyQtWebEngine).
4. Go to directory with "ChattyApp.py".
5. Run the ChattyApp using `python ChattyApp.py`
6. Have fun!
